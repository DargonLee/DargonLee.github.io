Grid布局入门

2020-03-07 · 8 min read

前言

给<div>这类块状元素元素设置display:grid或者给<span>这类内联元素设置display:inline-grid，Grid布局即创建！例如：

div {
    display: grid;
}

作用在grid容器上的CSS属性

  1. grid-template-columns和grid-template-rows
    这两个CSS属性用来对田地进行基本的划分，columns是列的意思，表示竖直方向的划分；rows是行的意思，表示水平方向的划分。
<div class="main">
        <div>1</div>
        <div>2</div>
        <div>3</div>
        <div>4</div>
        <div>5</div>
        <div>6</div>
    </div>

比如我们想把上面的布局分为3行3列

.main {
    grid-template-columns: 80px auto 100px;
    grid-template-rows: 25% 100px auto;
}

grid-template-columns后面3个值，表示分为了3列，从左往右每列的尺寸分别是80px，auto（自动）和100px；
grid-template-rows属性值含4个值，表示分为了4行，从上往下每行的高度分别是25%，100px，auto（自动）
<1> 这里分为多少行和多少列既可以用固定值也可以用百分比去写。例如下面这样

.container {
    grid-template-columns: 40% 60%;
    display: grid;
}

<2> 还可以使用弹性单位 fr，可以按比例分配容器的大小。总共分成 3 份，第一列占 1份，第二列占 2份：

fr单位是什么？
fr是单词fraction的缩写，表示分数。

.container {
    grid-template-columns: 1fr 2fr;
    display: grid;
}

分为三份 第一列占一份 第二列占两份

<3> 使用 minmax 设置长度范围，接受 2 个参数，分别表示最大值和最小值。下面设置列的最小宽度为 100px，最大宽度为 1fr。

.container {
    grid-template-columns: minmax(100px, 1fr) 1fr;
    display: grid;
}

<4>.使用 repeat 可以表示多个值，grid-template-columns：repeat(2, 1fr) 与grid-template-columns：1fr 1fr 等价。

.container {
    grid-template-columns: repeat(2, 1fr);
    display: grid;
}

grid-gap：间隙

grid-column-gap：列与列之间的间隙；grid-row-gap：行与行之间的间隙；grid-gap 为 grid-row-gap 和 grid-column-gap 的缩写。如果只设置一个值，表示行和列之间的间隙一样。
下面设置列与列的间隙为 10px，行与行的间隙为 5px：

.container {
    grid-template-columns: 40% auto;
    grid-template-rows: 60px 20%;
    display: grid;
    grid-gap: 5px 10px;
}

分块

就是在grid布局里面的盒子需要占多少行多少列，从第几行到第几行 从第几列到第几列。
例如下面代码我们分别去分块

<div class="main">
    <div class="huayuan">花园</div>
    <div class="caiyuan">菜园</div>
    <div class="house">房子</div>
    <div class="rest_area">休息区</div>
    <div class="guoyuan">果园</div>
    <div class="tree">树</div>
</div>
  1. 使用 grid 布局把整块地划分成 9 行 8 列的网格，分别使用属性 grid-template-columns 和 grid-template-rows。
.main {
    background-color: #eeeeee;
    display: grid;
    grid-template-columns: repeat(8, 1fr);
    grid-template-rows: repeat(9, 30px);
}
  1. 以网格线的编号来标记 house 的位置，行号开始为 3、结束为 8（用负数表示为 -3），列号开始为 3、结束为 7（用负数表示为 -3）。转换成 grid 布局中的属性 grid-row 和 grid-column，分别表示在网格布局中行与列的位置，这里用网格线的编号表示，开始结束使用 “/” 分开：
.house {
    background-color: cornflowerblue; 
    grid-row: 3 / -3;
    grid-column: 3 / -3;
}
  1. 再分花园
    行号开始为 8、结束为 10（用负数表示为 -1），列号开始为 3、结束为 7（用负数表示为 -3）：
.huayuan {
    background-color: brown;
    grid-row: 8 / -1;
    grid-column: 3 / -3;
}
  1. 休息区
    行号开始为 3、结束为 10（用负数表示为 -1），列号开始为 1、结束为 3：
.rest_area {
    background-color: darksalmon;
    grid-row: 3 / -1;
    grid-column: 1 / 3;
}
  1. 菜园和果园：
.caiyuan {
    background-color: cadetblue;
    grid-row: 1 / 3;
    grid-column: 1 / -3;
}
.guoyuan {
    background-color: hotpink;
    grid-row: 1 / -1;
    grid-column: -3 / -1;
}
  1. 设置间隙
    使用 row-gap 设置每列之间的间隙，column-gap 设置每行之间的间隙。
.main {
    background-color: #eeeeee;
    display: grid;
    margin: 10px 60px;
    grid-template-columns: repeat(8, 1fr);
    grid-template-rows: repeat(9, 30px);
    row-gap: 40px;
    column-gap: 60px;
}
  1. 树
.tree {
    background-color: green;
    grid-row: 10 / 11;
    grid-column: 1 / -1;
}

grid-template-areas

使用 grid-template-areas 来重新设计一个与上面一模一样的图。这个属性可以根据区域名称把某些 item 组合起来，区域名称需要通过 item 属性 grid-area 来定义。使用的时候每个值需要使用空格隔开，而且必须组成一个矩形，对于不能组成矩形的属性将不起作用。它的值可以是：

  1. 通过属性 grid-area 定义名字；
  2. 使用“.”指定一个空的 cell;
  3. none 不定义区域。
    下面通过这个属性实现与上面一样的效果：
<style>
.main {
    background-color: #eeeeee;
    display: grid;
    grid-template-columns: repeat(8, 1fr);
    grid-template-rows: repeat(9, 40px);
    grid-auto-rows: 40px;
    row-gap: 20px;
    column-gap: 20px;
    grid-template-areas: 
        "caiyuan caiyuan caiyuan caiyuan caiyuan caiyuan guoyuan guoyuan"
        "caiyuan caiyuan caiyuan caiyuan caiyuan caiyuan guoyuan guoyuan"
        "rest_area rest_area house house house house guoyuan guoyuan"
        "rest_area rest_area house house house house guoyuan guoyuan"
        "rest_area rest_area house house house house guoyuan guoyuan"
        "rest_area rest_area house house house house guoyuan guoyuan"
        "rest_area rest_area house house house house guoyuan guoyuan"
        "rest_area rest_area huayuan huayuan huayuan huayuan guoyuan guoyuan"
        "rest_area rest_area huayuan huayuan huayuan huayuan guoyuan guoyuan"
        ". . tree tree tree tree . .";
}
.huayuan {
    background-color: brown;
    grid-area: huayuan;
}
.caiyuan {
    background-color: cadetblue;
    grid-area: caiyuan;
}
.house {
    background-color: cornflowerblue;
    grid-area: house;
}
.rest_area {
    background-color: darksalmon;
    grid-area: rest_area;
}
.guoyuan {
    background-color: hotpink;
    grid-area: guoyuan;
}
.tree {
    background-color: green;
    grid-area: tree;
}
</style>

grid对齐

  1. justify-items（容器属性）: 定义 grid-item 在 grid-area（布局区域）水平方向的（左右）对齐方式：

  2. justify-content: 它的意思是 justify-content 这个属性用来给内容周围分配空间，在 flex 容器中它按照主轴进行分配，在 grid 容器中按照 inline axis 进行分派。

  3. justify-self：表示 item 在布局区域水平方向的对齐方式。
    在「竖直轴」上的对齐方式主要通过align-content, align-items and align-self 来设置，都以 align- 开头。

  4. align-items：这个属性适用于 FlexBox 和 grid 布局，在 FlexBox 布局中它表示 items 在纵轴上的对齐方式；在grid布局中，表示item在 grid area 竖直方向上对齐方式。

  5. align-content: 如果所有的 items 尺寸和小于 grid 容器的尺寸，可以通过 align-content 设置内容在竖直轴上的对齐方式：

  6. align-self: 表示自己在竖直轴上的对齐方式。

1.place-content: 是 align-content 和 justify-content 的缩写。为方便记忆，可以联想到“Angelababy”，多念几次，是不有点像 “aj”。
2.place-items: 它是 align-items 和 justify-items 的缩写方式。
